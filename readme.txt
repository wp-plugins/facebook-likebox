== Description ==

Let people share pages and content from your site back to their Facebook profile with one click, so all their friends can read them.

Facebook Likebox is important part of social optimization, that's why a lot of people use Facebook like box on their websites. 


= Features =

* User friendly and easy to use.
* Tested with other popular plugins.
* Ability to set connections number.
* Ability to set Facebook like box width.
* Ability to set Facebook like box height.
* Ability to show/hide Facebook logo bar.
* Ability to change Facebook like box language.
* Ability to add Facebook like box in posts/pages using shortcode button.
* Ability to add Facebook like box Popup.

== Installation ==

### Performing a new installation for Gallery Bank.

After downloading the ZIP file,  

1. Log in to the administrator panel.  
2. Go to Plugins Add > New > Upload.  
3. Click "Choose file" ("Browse") and select the downloaded zip file.  

**For Mac Users**  
> * Go to your Downloads folder and locate the folder with the plugin. 
> * Right-click on the folder and select Compress. 
> * Now you have a newly created .zip file which can be installed as described here.*  
> * Click "Install Now" button.  
> * Click "Activate Plugin" button for activating the plugin.
